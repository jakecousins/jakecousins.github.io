DIY Walkie Talkie

Problem Description:

Our team was tasked with creating some sort of device based on preset a list of certain ideas. We chose to try and create a DIY Walkie Talkie out of very basic components. This device could use some small invididual components on their own PCB but we were not to create one singular PCB with all components and connections, everything had to be done on a breadboard.

Design Features

  • Speaker
  • Microphone
  • Volume Adjustment Knob
  • Freqency Adjustmnet Not
  • Antenna

It did not have the most robust range or the best audio, but given the limitations imposed on us and the very basic components used we were quite happy with the result of this project.
